初识 Go 程序 #

第一个 Go 程序 #

go
// 定义一个包, 包名为main, main 是可执行程序的包名, 所有的 Go 源程序文件头部必须有一个包声明语句, Go 通过宝来管理命名空间
package main

// import 引入一个外部包 fmt, 可以是标准库的包, 也可以是第三方或自定义的包, fmt 是标准输入/输出包
import "fmt"

// 使用 func 关键字声明定义一个函数, 函数名为 main, mian 代表 Go 程序入口函数
func main() {
	// 调用 fmt 包里面的 Printf 函数, 函数参数是一个字符串字面量, 在标准输入里面打印一句话 "Hello, world"
	fmt.Println("Hello, world")
}

源代码的特征 #

  • 源程序以.go为后缀
  • 源程序默认为UTF-8编码
  • 标识符区分大小写
  • 语句结尾分号可以省略
  • 函数以func 开头, 函数体开头的 "{" 必须在函数头所在行尾部, 不能单独起一行
  • 字符串字面量使用 "" (双引号)括起来
  • 调用包里的方法通过点 . 访问符
  • main 函数所在的包名必须是 mian

Go 词法单元 #

token #

​ token 是构成源程序的基本不可再分割的单元, 编译器编译源程序的第一步就是将源程序分割为一个个独立的 token, 这个过程就是词法分析

  • 操作符: 操作符就是一个天然的分隔符, 同时其自身也是一个 token
go
sum := a + b

:= 和 + 既是分隔符, 也是 token

  • 纯分隔符: 其实本身不具备任何语法含义, 只作为其他 token 的分割功能(空格, 制表符, 换行符, 回车符)
go
package    main

这是一个包声明的语句, package 和 main 之间可以有任意多个空格或制表符, 但 Go 编译器 会将其作为一个分隔符处理

标识符 #

​ 编程语言的标识符用来表示变量, 类型, 常量等语法对象的符号名称

​ Go 的标识符构成规则: 开头一个字符必须是字母或者下划线

常量 #

go
package main

import "fmt"

func main() {
	const (
		a = 1 << iota      // iota = 0
		b = 1 << iota      // iota = 1
		c = 3              // iota = 2
		d = 1 << iota      // iota = 3 计算方式 1 * 2^3
		e = 4              // iota = 4
		f = 2 << iota * 10 // iota = 5 计算方式 2 * 2^5 * 10
	)
  // iota 每次都是从 0 开始
  const (
    g, h = iota, iota + 4
  )
	fmt.Printf("%d %d %d %d %d %d", a, b, c, d, e, f) // 1 2 3 8 4 640
	fmt.Printf("%d %d", g, h) // 0 4
}

基本数据类型 #

布尔类型 #

  • 布尔类型只有两个值: true 和 false
  • 布尔型数据 不能 和整型数据相互装换
  • 声明的布尔型变量不指定初始值, 则默认为false

整型 #

  • 不同类型的整型 必须 镜像强制类型转换
  • 支持算数运算和位操作, 算数表达式和位操作表达式的结果还是 整型

浮点型 #

  • 浮点数字面量被总动类型推断为 float64 类型
  • 计算机很难进行浮点数精确表示和存储, 不应该使用 == 或 != 进行比较, 高精度应该使用 math 标准库

复数类型 #

  • 复数使用两个浮点数表示, 一个表示实部, 一个表示虚部
  • 构造一个复数 complex(x, y)
  • 返回复数实部 real(v)
  • 返回复数虚部 image(v)

字符串 #

  • 字符串是常量, 可以通过类似数组的索引访问字节单元, 但是不能修改某个字节的值

  • 字符串转换为切片 []byte() 要慎用, 尤其数据量较大时 (每次都会复制内容)

  • 字符串尾部不包含 Null

  • 字符串类型底层实现

    go
    // runtime/string.go
type stringStruct struct {
	str unsafe.Pointer // 指向底层字节数组的指针
	len int            // 字节数组长度
}

  • 字符串切片操作返回的子串仍然是 string

    go
    package main

import "fmt"

func main() {
	s := "hello world"

	a := s[:2]	// he
	b := s[2:] 	// llo world
	c := s[1:4] // ell
	fmt.Printf("0到2----%s \n2到最后----%s \n1到4----%s", a, b, c)
}

  • 字符串可以转换为字节数组, 也可以转换为 Unicode 的字数组

    go
    package main

import "fmt"

func main() {
	s := "hello world"

	a := []byte(s)
	b := []rune(s)
	fmt.Printf("字节数组----%d \n字数组------%d", a, b)
}

  • 字符串的运算

    go
    package main

import "fmt"

func main() {
	a := "hello"
	b := "world"

	c := a + b

	d := len(c)

	fmt.Printf("字符串拼接-----%s \n字符串长度-----%v", c, d)
}

指针 #

  • 在赋值语句中, *T 出现在 "=" 左边表示指针声明, *T 出现在 "=" 右边表示取指针指向的值

    go
    package main

	a := 11
	b := &a
	fmt.Printf("内存地址----%b \n获取值------%v", b, *b)
}

  • 结构体指针访问结构体字段仍然使用"."点操作符

    go
    package main

import "fmt"

type User struct {
	name string
	age  int
}

func main() {
	a := User{
		name: "tjl",
		age:  24,
	}

	b := &a
	c := a
	fmt.Printf("指针访问结构体字段----%v \n正常访问结构体字段----%v", b.name, c.name) // tjl tjl
}

  • Go 不支持指针的运算

  • 函数中允许返回局部变量的地址

    go
    package main

import "fmt"

func sum(a, b int) *int {
	sum := a + b
	return &sum
}
func main() {
	a := sum(1, 2)
	fmt.Printf("内存地址--%v \n数值------%v", a, *a)

数组 #

  • 数组初始化

    go
    package main

import "fmt"

func main() {
	a := [3]int{1, 2, 3}            // 指定长度和初始化字面量
	b := [...]int{1, 2, 3, 4, 5, 6} // 不指定长度, 但是由后面的初始化列表数量来确定其长度
	c := [3]int{1: 1, 2: 2}         // 指定总长度, 并通过索引值进行初始化, 没有初始化元素时使用类型默认值
	d := [...]int{0: 1, 2: 2}       // 不指定总长度, 通过索引值进行初始化, 数组长度由最后一个索引值确定, 没有指定索引的元素被初始化为类型的零值
	fmt.Printf("%v\n%v\n%v\n%v", a, b, c, d)
  // [1 2 3]
  // [1 2 3 4 5 6]
  // [0 1 2]
  // [1 0 2]
}

  • 数组的特性

    1. 数组创建完长度就固定了, 不可以再追加元素
    2. 数值的值类型的, 数组赋值或作为函数参数都是值拷贝
    3. 数组长度是数组类型的组成部分, [10]int 和 [20]int 表示不同类型
    4. 可以根据数组创建切片

  • 数组的相关操作

    1. 数组元素访问

      go
      package main

import "fmt"

func main() {
	a := [3]int{1, 2, 3}
	b := a[1]
	fmt.Println(b) // 2
}

    2. 数组长度

      go
      package main

import "fmt"

func main() {
	a := [3]int{1, 2, 3}
	b := len(a)
	fmt.Println(b) // 3
}

切片 #

数据类型 slice 切片, 这是一种变长数组, 其数据结构中有指向数组的指针, 所以是一种 引用类型

go
type slice struct {
	array unsafe.Pointer // 指针地址
	len   int            // 切片元素数量
	cap   int            // 切片底层数组容量
}
  • 创建切片
    1. 由数组创建
    2. 通过内置函数 make 创建
  • 切片支持的操作
    1. 内置函数 len(v) 返回切片长度
    2. 内置函数 cap(v) 返回切片底层数组容量
    3. 内置函数 append(v, arr) 对切片追加元素
    4. 内置函数 copy(v1, v2) 赋值一个切片

map #

  • map 的创建

    1. 使用字面量创建

      go
      package main

import "fmt"

func main() {
	ma := map[string]int{"a": 1, "b": 2}
	fmt.Println(ma["a"]) // 1
}

    2. 使用内置 make 函数创建

      go
      package main

import "fmt"

func main() {
	/*
		make(map[KeyType]ValueType, len)
		KeyType: 键类型
		ValueType: 值类型
		len: 长度
	*/
	ma := make(map[int]string, 10)
	ma[1] = "tom"
	fmt.Println(ma[1]) // tom
}

  • map 支持的操作

    1. map 的单个兼职访问格式为 mapName[key]
    2. 可以使用 range 遍历 map 类型变量, 但是不保证每次迭代元素的顺序
    3. 删除 map 中的某个键值 delete(mapName, key)
    4. 可以使用 len() 返回 map 中键值对数量

  • 注意

    • Go 内置的 map 不是并发安全的,并发安全的 map 可以通过标准包 sync 中的 map
    • 不要直接修改 map value 内某个元素的值

控制结构 #

程序指令都是线性地存放在存储器上. 程序执行从本质上来说就是两种模式: 顺序和跳转

  • 顺序就是按照程序指令在存储器上的存放顺序逐条执行
  • 跳转就是遇到跳转指令就跳转到某处继续线性执行

if 语句 #

  • if 后面的条件判断子句不需要用小括号括起来
  • { 必须放在行尾, 和 if 或 if else 放在一行
  • if 后面可以带一个简单的初始化语句, 并以分号分隔
  • Go 语言没有三元运算符 (a > b ? a : b)
  • if 分支语句遇到 return 后直接返回, 遇到 break 则跳过 break 下方的 if 语句块

switch 语句 #

  • switch 和 if 语句一样, switch 后面可以带一个可选的简单的初始化语句
  • switch 后面的表达式也是可以选的
  • switch 条件表达式可以是任意支持相等比较运算的类型变量
  • 通过 fallthough 语句来强制执行下一个 case 子句
  • switch 支持 default 语句, default 语句可以放任意位置, 并不会影响 switch 判断
  • switch 和 .(type) 结合可以进行类型的查询
  • select 是一种类似 switch 语法结构的分支语句

for 语句 #

  • for 循环语句

    go
    for init; condition; post {  }

  • 类似其他语音的 while 循环语句

    go
    for condition {  }

  • while(1) 死循环语句

    go
    for {  }
go
package main

import "fmt"

func main() {
	// 遍历数组
	a := [...]int{1, 2, 3, 4, 5}
	for index, value := range a {
		fmt.Printf("索引--%v值--%v\n", index, value)
	}
	// 遍历map
	b := map[string]int{
		"a": 1,
		"b": 2,
	}
	for key, value := range b {
		fmt.Printf("键--%v值--%v\n", key, value)
	}
	// 遍历切片
	c := make([]int, 4)
	for index, value := range c {
		fmt.Printf("索引--%v值--%v\n", index, value)
	}
}

标签和跳转 #

goto, break, continue 语句的跳转

  • goto

    • goto 语句只能在函数内跳转
    • goto 语句不能跳过内部变量声明语句
    • goto 语句只能跳到同级作用域或者上层作用域, 不能跳到内部作用域内
    go
    package main

import "fmt"

func main() {
	a := 1
	if a > 0 {
		goto label
	}
	fmt.Println("不走这里")
label:
	fmt.Println("走这里")
}

  • break

    用于函数内跳出 for, switch, select 语句的执行

    • 单独使用
    • 和标签一起使用
    go
    package main

func main() {
L1:
	for i := 0; ; i++ {
		for j := 0; ; j++ {
			if i >= 5 {
				// 跳出到 L1 标签所在的 for 循环
				break L1
			}
			if j > 10 {
				// 默认跳出离 break 最近的内层循环
				break
			}
		}
	}
}

  • continue

    用于跳出 for 循环的本次迭代, 跳到 for 循环的下一次迭代的 post 语句处执行

    • 单独使用
    • 和标签一起使用
